# HLS Видео

Эта папка содержит видео в формате HLS (HTTP Live Streaming) для оптимизированной потоковой передачи.

## Что такое HLS?

HLS - это протокол потоковой передачи видео, разработанный Apple. Он разбивает видео на небольшие сегменты (.ts файлы) и создает плейлист (.m3u8), который управляет воспроизведением.

### Преимущества HLS:
- **Адаптивный битрейт**: автоматическая подстройка качества под скорость интернета
- **Быстрая загрузка**: видео начинает воспроизводиться быстрее
- **Меньшая нагрузка**: сегменты загружаются по мере необходимости
- **Широкая поддержка**: работает на всех современных браузерах и устройствах

## Конвертация видео в HLS

Для конвертации нового видео используйте команду:

```bash
ffmpeg -i input.mp4 \
  -c:v libx264 -c:a aac \
  -hls_time 6 \
  -hls_playlist_type vod \
  -hls_segment_filename "hls/videoname_%03d.ts" \
  -start_number 0 \
  hls/videoname.m3u8
```

### Параметры:
- `-i input.mp4` - входной файл
- `-c:v libx264` - кодек видео (H.264)
- `-c:a aac` - кодек аудио (AAC)
- `-hls_time 6` - длительность каждого сегмента в секундах
- `-hls_playlist_type vod` - тип плейлиста (VOD = Video On Demand)
- `-hls_segment_filename` - шаблон имени для сегментов
- `-start_number 0` - начальный номер сегмента

## Использование в коде

После конвертации используйте видео в компоненте:

```tsx
import srcVideoHls from 'assets/videos/hls/videoname.m3u8';

<VideoPlayer
  videoSources={[{ src: srcVideoHls, type: 'application/x-mpegURL' }]}
/>
```

## Текущие файлы

- `hero.m3u8` - плейлист для главного видео
- `hero_000.ts`, `hero_001.ts`, `hero_002.ts` - сегменты видео

## Примечания

- Файлы .ts в этой папке - это видео сегменты, а не TypeScript файлы
- Не удаляйте .m3u8 файл без соответствующих .ts сегментов
- При деплое убедитесь, что сервер правильно отдает MIME-типы для .m3u8 и .ts файлов
